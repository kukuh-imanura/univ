TUTORIAL MEMBUAT AKUN GITHUB
==============================

1.  Buka https://www.github.com
2.  Klik pada tulisan "Create an account"
3.  Masukkan data-data yang terdiri dari : 
    - Email
    - Password
    - Username
4.  Kemudian akan ada penawaran untuk menerima "Announcement and Product Update" via Email.
    Teman-teman dapat memilih "Y" atau "N" sesuai keinginan
5.  Setelah itu, teman-teman akan diminta untuk verifikasi akun dengan mengisi "Captcha"
6.  Akhirnya, teman-teman hanya perlu untuk memasukkan kode yang sudah dikirimkan melalui Email
7.  Setelah itu akan ada beberapa pertanyaan-pertanyaan untuk personalisasi pengguna
    Teman-teman dapat melewatinya dengan meng-klik "Skip" pada bagian bawah layar
